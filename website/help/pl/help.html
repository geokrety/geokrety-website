O GeoKretach

GeoKrety (z greckiego geo = 'ziemia' oraz polskiego: krety) to serwis podobny do TravelBug(TM) lub GeoLutins, chcący śledzić obiekty pozostawione w skrzynkach geocache. Ludzie przenoszą te zarejestrowane obiekty (nazywane GeoKretami) ze skrzynki do skrzynki i zapisują każdy jego ruch za pomocą tego serwisu. Podróż każdego GeoKreta jest następnie pokazana na mapie.


Kret

Jak się wymawia GeoKrety? ;)

GeoKretów się nie wymienia, nie ma potrzeby umieszczania czegoś w skrzynce aby wziąć GK, nie powinieneś także brać czegoś w zamian za GK. Jeżeli chcesz wziąć coś ze skrzynki, powinieneś coś zostawić w zamian. Możesz wtedy też odłożyć GK.

Ten serwis jest całkowicie bezpłatny (w przeciwieństwie do TravelBug) i jest łatwy w obsłudze. GeoKret może być wszystkim, w większości przypadków jest małą maskotką.

Please see (and edit, if necessary) the article about geokrety at Wikipedia.

Zrób własnego Geokreta

Obejrzyj filmik instruktażowy o rejestracji i tworzeniu własnych GeoKretów.

  1. Posiadaj konto na serwisie GeoKrety.org
  2. Zarejestruj GeoKreta i zapisz jego Kod Śledzenia GeoKreta (np. XF3ACS) wygenerowany przez system. Ten kod jest tajny i tylko ty i ludzie którzy znajdą twojego GeoKreta powinni go znać.
  3. Wydrukuj etykietę z nazwą GeoKreta, Kodem Śledzenia, Kodem Odniesienia (np. GK003F) i jego właścicielem. Przyczep tą etykietę do GeoKreta (np. maskotki)

    Przykłady gotowych GeoKretów

Następnie musisz:

  1. Odłożyć swojego GeoKreta do skrzynki, i zarejestrować jego położenie w serwisie
  2. Cieszyć się podczas oglądania podróży twojego GeoKreta!
Typy GeoKretów
  • Tradycyjny - maskotka lub inny podobny przedmiot ze znacznikiem
  • Książka/CD/DVD - gdy chcesz śledzić książki, płyty CD/DVD wsadzone do skrzynek. Przyklej numer GeoKreta i krótką instrukcję na tyle książki/opakowania płyty.
  • Człowiek - kiedy chcesz śledzić samego siebie i pisać coś w stylu GeoBloga
  • Moneta (toczygrosz) - normalny GeoKret w formie monety
  • Krecia Poczta - normalny GeoKret w formie pocztówki/poczty

    Krótki opis:

    1. Weź kartkę (może być pocztówka lub kartka domowej roboty) i napisz kilka zdań dla odbiorcy.
    2. Adres odbiorcy to nazwa użytkownika, region (kraj, województwo, miasto) i docelowa skrzynka (najlepiej ze wszystkimi możliwymi punktami orientacyjnymi - kod GC, kod OC itp.).
    3. Stwórz GeoKreta z typem "KretyPost", wydrukuj znaczek (zobacz przykłady) z kodem śledzenia i umieść znaczek na kartce. (możesz wygenerować prosty znaczek w generatorze etykiet na stronie geokreta)
    4. (nieobowiązkowe:) Zalaminuj gotową kartkę by chronić ją podczas podróży.
    5. Odłóż kartkę do pobliskiej skrzynki, oznacz lokalizację GeoKreta i czekaj... :-)
    6. (nieobowiązkowe:) Poinformuj odbiorcę, on też zapewne chciałby obserwować tego GeoKreta ;)

Jak rejestrować GeoKrety w skrzynkach geocaching.com

GC waypoints are not fully supported by GeoKrety (Licensee shall not remarket, resell, and/or redistribute the Data or any derived portion(s) of the Data in its digital form to any third party whatsoever... from waypoint license agreement). However you can easily use GeoKrety with GC caches. There are two options:

  • The easiest and recommended way is to use our GeoKrety Toolbox (GKT) - a greasemonkey script (written and maintained by simor). Currently it can be used in the following browsers:

    - Mozilla FireFox (from version 3.5), you also need Greasemonkey Add-on. Install GKT for Firefox here.

    - Google Chrome. Install GKT for Chrome here.

    You also need an account on geokrety.org. After installing GKT in your browser, you will see an extra [ GeoKrety.org ] panel on the right hand side of every GC cache page. From the list of GeoKrety pick the one you wish to drop and click "Drop selected GK" - you will be redirected to the standard logging page with both the GC Code and coordinates already filled in. Check the date, add a comment and you're done.

    Here is how GKT looks like:

  • Manual way. Go to Log a GeoKret page and under the section New location enter the following:
    1. in the waypoint field, enter the GC Code of the cache (eg. GC1AQ2N).
    2. in the lat/lon field, enter the coordinates of the cache (you can copy-paste it directly from cache page or use other acceptable formats)

Akceptowalne formaty współrzędnych geograficznych

You can enter lat/lon coordinates in different formats:

  • degrees (DD.dddddd°)
    • 	52.205205 21.190891
	52.205205/21.190891
	N 52.205205 E 21.190891
	+52.205205 -21.190891
  • degrees, minutes (DD°MM.mmm’)
    • 	52 12.3123 -21 11.45345
	N52 12.3123 W21 11.45345
	+52 12.3123 -21 11.45345
	N 52° 10.369' W 021° 01.542'
  • degrees, minutes, seconds (DD°MM’SS.s”)
    • 	52 12 18.74 -21 11 27.21
	N52° 12' 18.74", W21° 11' 27.21"
	52° 12' 18.74", -21° 11' 27.21"
Ikonologia (jak edytować, usuwać logi, przesłać zdjęcia itp.)
Ikona Opis
edit - an email address, description, log entry

upload photo - attach a photo to a log entry or your profile

report missing - you may report a GeoKret missing if you have visited the cache where the GeoKret should be but it wasn't there
Numer referencyjny (Reference Number) i kod śledzenia (Tracking Code)
  • Reference Number e.g. GK003F. It is a public number that may be used to refer a GeoKret. It consists of 'GK' prefix and 4 digits in hexadecimal system. You can use it to search a specific GeoKret or to link to this GeoKret. It is visible to everyone.
  • Tracking Code eg XF3ACS. It is a secret identification code which is required to post logs such as "Grab", "Drop" and others for a GeoKret. Only the owner and geocachers who found the GeoKret should know this code. It is not recommended to post it anywhere. Of course this code must be written on GeoKret's tag to allow finders to log it.
Wysokość i kod kraju

Thanks to SRTM and GTOPO30 elevation data and lat/lon to country code conversion (geonames), each trip point has assigned an altitude and country code (a flag). According to that data, the altitude profile is drawn. Above data and plots are updated every 10-15 minutes.

Kody QR

For easy logging dropping or grabbing during your trip, I've added the possibility of attaching QR code on the GK label. QR Code is a kind of bar code, which can be "scanned" by your mobile phone, which will redirect you to the proper page at geokrety.org; you will not have to enter the tracking code, everything will be encrypted in the QR. Read more about QR Code.

The QR code with encrypted address of geokrety.org page.

When you find a GeoKret with QR code on the label you can:

  1. Take a photo of the code with your mobile phone
  2. Decode it and click on the url
  3. Log the grabbing or hiding the GeoKret. The tracking code will be indicated by the url.

How to decode

You can use the free software for your mobile phone, eg: Barcode Scanner
