Acerca do GeoKrety

GeoKrety (do Grego geo = ‘terra’, e Polaco krety = ‘moles’) é um serviço similar a TravelBug(TM) or GeoLutins, que visa seguir objectos que deixa nos recipientes das geocaches. As pessoas movem esses objectos registados (chamados GeoKrety) de cache para cache e registam cada movimento com este serviço. A viagem de cada GeoKret é então apresentada num mapa.


Uma toupeira

Como pronunciar geokrety? ;)

GeoKrety não é um item de troca, não precisa colocar nada na cache para tirar um GK, e não deve tirar nada de uma cache por troca com um GK. Se quiser tirar algo de uma cache, deve fazer uma troca e pode colocar o GK.

O serviço é absolutamente gratuito (ao contrário do TravelBug) e fácil de usar. Um GeoKret pode ser qualquer coisa, mas na maioria dos casos é um pequeno boneco.

Por favor veja (e edite, se necessário) o artigo sobre geokrety na Wikipedia.

Faça o seu próprio GeoKret

Assista a uma screencast instrucional sobre como se registrar e fazer seu próprio GeoKret.

  1. Possui uma conta no serviço GeoKrety.org
  2. Registre o GeoKret e anote no GeoKret (exemplo XF3ACS) gerado pelo sistema. Este código é secreto e somente você e pessoas que encontram este GeoKret devem conhecê-lo.
  3. Imprima uma etiqueta com o nome do GeoKret, código de rastreamento, número de referência (por exemplo, GK003F) e proprietário nele. Anexar esse rótulo ao GeoKret (por exemplo, uma puppet)

    Exemplos de GeoKrety prontos

Então você tem que:

  1. Coloque seu GeoKret em um cache e registre sua localização no serviço
  2. Divirta-se assistindo à viagem do seu GeoKret!
Tipos de GeoKret
  • Tradicional - um pequeno fantoche com uma tag
  • Um livro/CD/DVD/... - quando você deseja rastrear livros, CDs, DVDs caiu no cache de containers. Basta colocar um número GeoKret e instruções curtas no verso de uma capa.
  • Um ser humano - quando você quer se acompanhar e escrever algo como GeoBlog
  • Uma moeda (ou: gecoins; polido: toczygrosze) - GeoKret normal, mas em forma de uma moeda
  • KretyPost - geoKret regular, mas em forma de cartão/correio

    Breve descrição:

    1. Pegar um cartão (que pode ser um cartão-postal, mas também um cartão home) e escrever algumas linhas para o receptor nele.
    2. Como endereço do destinatário o apelido do destinatário, a região (país, distrito, cidade) e dá um cache de destino (melhor com todos os waypoints disponíveis - GC-code, OC-code, etc.).
    3. Create a GeoKret with type "KretyPost", print a stamp (see examples) with the tracking code and put this stamp on the postcard. (você pode gerar um carimbo simples no gerador de rótulo na página geokret)
    4. (opcional:) Lamine o cartão postal concluído para protegê-lo durante a jornada.
    5. Coloque o cartão em um cache perto do seu local inicial, registre o GeoKret como caiu e espere... :-)
    6. (opcional:) Informe o destinatário, ele provavelmente vai gostar de observar este geokret

Tipos de registo
Ícone Descrição

No entanto, você pode facilmente usar GeoKrety com cache de coleção. Existem duas opções:

    - Mozilla FireFox (da versão 3.5), você também precisa do Greasemonkey Add-on. Instale o GKT para o Firefox aqui.

    - Google Chrome. Instale o GKT para o Chrome aqui.

Da lista de GeoKrety escolha a que você deseja remover e clique em "Drop selecionado GK" - você será redirecionado para a página de log padrão com o código GC e as coordenadas já preenchidas. Formatos aceitáveis para coordenadas geográficas

Você pode inserir coordenadas de lat/lon em diferentes formatos:

  • Graus (DD.dddddd°)
    • 	52.205205 21.190891
	Common 52.205205/21.190891
	Common 52.205205 E 21.190891
	Common +52.205205 -21.190891
  • graus, minutos (DD°MM.mmm')
    • 	52 12.3123 -21 11.45345
	Common N52 12.3123 W21 11.45345
	Common +52 12.3123 -21 11.45345
	Common N 52° 10.369' W 021° 01.542'
  • graus, minutos, segundos (DD°MM'SS.s")
    • 	52 12 18.74 -21 11 27.21
	£N52° 12' 18.74", W21° 11' 27.21"
	thanking 52° 12' 18.74", -21° 11' 27.21"

Número de Referência e Código de Seguimento
  • Número de referência por exemplo, GK003F. É um número público que pode ser usado para indicar um GeoKret. Ele consiste no prefixo 'GK' e 4 dígitos no sistema hexadecimal. Você pode usá-lo para pesquisar um GeoKret específico ou vincular a este GeoKret. Ele é visível para todos.
  • Tracking Code eg XF3ACS. É um código de identificação secreto que é necessário para postar logs como "Grab", "Drop" e outros para um GeoKret. Somente o proprietário e geocachers que encontraram o GeoKret devem saber esse código. It is not recommended to post it anywhere. É claro que esse código deve ser escrito na tag do GeoKret, para permitir que os buscadores o registrem.
Altitude e código do país

Graças a SRTM e GTOPO30 dados de elevação e lat/lon para conversão de código do país (geonames), em cada ponto de viagem foi atribuído uma altitude e um código de país (uma bandeira). De acordo com esses dados, o perfil de altitude está gravado. Os dados e lotes acima são atualizados a cada 10-15 minutos.

Código QR

Para facilitar o log dropando ou agitando durante sua viagem, adicionei a possibilidade de anexar o código QR no rótulo do GK. Código QR é um tipo de código de barras, que pode ser "escaneado" pelo seu telefone celular. que irá redirecioná-lo para a página adequada no geokrety. rg; você não precisará inserir o código de rastreamento, tudo será criptografado no QR. Leia mais sobre o Código QR.

O código QR com endereço criptografado da página geokrety.org.

Quando você encontrar um GeoKret com código QR na etiqueta, você pode:

  1. Tire uma foto do código com seu telefone celular
  2. Decodifique-o e clique na url
  3. Registre o capturamento ou esconda o GeoKret. O código de rastreamento será indicado pela url.

Como decodificar

Você pode usar o software gratuito para seu celular, por exemplo: Scanner de Código de Barras

Protecção de dados pessoais

Seu endereço de e-mail não será transferido para terceiros. Sua senha é criptografada unidiretamente, por isso não pode ser revelada.

Política de cookies do GK

Este site usa cookies para armazenar alguma informação necessária para utilizar o site de forma conviniente. Nós usamos cookies para: 1) Armazenar informações de login (verificar logado no usuário) e 2) Armazenar preferências de idioma. Ambos são absolutamente inofensivos . É isso! :)

No entanto, se você quiser, você pode alterar suas configurações de cookies a qualquer momento em seu navegador.

HTTPS e SSL

Você pode usar conexão segura (https) com o nosso servidor: https://geokrety.org ou usar um ícone do padlock no final da página.
