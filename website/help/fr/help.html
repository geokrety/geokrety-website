À propos de GeoKrety.org

GeoKrety (du Grec geo = ‘terre’ et du Polonais krety = ‘taupe’) est un service, similaire à TravelBug(TM) ou GeoLutins, permettant de suivre des objets voyageurs que vous laissez dans des géocaches. Les autre joueurs déplacent ces objets enregistrés (appelés GeoKrety) d'une cache à l'autre et signalent chaque mouvement sur ce site Web. Le trajet de chaque GeoKret peut alors être suivi sur une carte.


Une taupe

Comment prononcer 'GeoKrety' ? ;)

Les GeoKrety ne sont pas des objets d'échange. Vous n'êtes pas obligé de déposer un objet dans une cache pour prendre un GeoKret. Si vous voulez prendre quelque chose dans une cache alors faites un échange puis déposez ou prenez un GeoKret. Le service est totalement gratuit (à l'inverse des TravelBug) et facile à utiliser. Un GeoKret peut être n'importe quoi, mais dans la plupart des cas, c'est une petite figurine.

Veuillez consulter (et modifier, si nécessaire) l’article sur les GeoKrety de Wikipedia.

Créer votre propre GeoKret

Regardez une vidéo sur la création de votre propre GeoKret.

  1. Ouvrez un compte sur GeoKrety.org
  2. Enregistrez le GeoKret et notez le Tracking Code du GeoKret (par exemple XF3ACS) généré par le système. Ce code est secret et seuls vous et les personnes qui trouvent le GeoKret devraient le connaître.
  3. Imprimer une étiquette avec le nom de votre GeoKret, le Tracking Code, le Numéro de Référence (par exemple GK003F) et le nom du propriétaire (vous). Attachez l'étiquette au GeoKret (ex.

Les différentes catégories de GeoKrety
  • Traditionnel - Une petite figurine avec son étiquette
  • Un livre/CD/DVD/… - Lorsque vous souhaitez effectuer le suivi de livres, CDs, DVDs déposés dans des caches. Collez simplement les identifiants du GeoKret et de courtes instructions sur l'envers de la pochette.
  • Un humain - Si vous souhaitez suivre vos déplacements.
  • Une pièce de monnaie (en anglais geocoin, en polonais toczygrosze) - Une pièce sur laquelle vous avez gravé ou ajouté son numéro de suivi
  • KretyTimbre - GeoKret ordinaire, mais sous forme d'une carte postale/courrier

    Principe de fonctionnement :

    1. Prenez une carte (carte postale traditionnelle ou carte de votre propre conception) et écrivez quelques lignes à l'attention du destinataire.
    2. Sur la carte précisez l'adresse de destination sous la forme : pseudo du destinataire, destination (continent, pays, région, ville) et une cache cible (coordonnées GPS ou mieuxun WayPoint valide comme un code GeoCaching, un code OpenCaching, une ciste…).
    3. Sur GeoKrety.org, créez un GeoKret du type "KretyPost", imprimez un timbre avec le Tracking Code et collez-le sur la carte (voir quelques exemples de timbres ici). (vous pouvez générer des timbres dans le générateur d'étiquette sur la page du GeoKret)
    4. (optionnel) Plastifiez la carte pour la protéger durant son voyage.
    5. Déposez la carte dans une cache proche de votre domicile, revenez sur GeoKrety.org pour signaler ce GeoKret comme 'déposé' et attendez… :-) :-) :-) :-)
    6. (optionnel) Prévenez le destinataire pour qu'il puisse suivre la progression de la carte et surtout afin qu'il récupère la carte lorsqu'elle sera arrivée dans la cache indiquée comme destination.

Les types de signalements
Icônes Description
ou J'ai déposé un GeoKret : lorsque vous déposez un GeoKret dans une cache.
ou J'ai pris un GeoKret : lorsque vous prenez un GeoKret dans une cache afin de le déposer rapidement dans une autre cache.
ou J'ai rencontré un GeoKret : lorsque vous découvrez un GeoKret et que vous le laissez dans la cache.
ou Un GeoKret a visité la cache : lorsqu'un GeoKret est en votre possession, que vous trouvez une cache mais qui vous n'y déposez pas le GeoKret.
Commentaire : lorsque vous souhaitez laisser un message sur un GeoKret :)
Hibernation / archive : lorsqu'un GeoKret n'a pas été vu depuis longtemps.

Les formats de coordonnées géographiques acceptables

Vous pouvez entrer les coordonnées lat/lon dans différents formats :

  • Degrés (DD.dddddd°)
    • 	52.205205 21.190891
	52.205205/21.190891
	N 52.205205 E 21.190891
	+52.205205 -21.190891
  • Degrés, minutes (DD°MM.mmm')
    • 	52 12.3123 -21 11.45345
	N52 12.3123 W21 11.45345
	+52 12.3123 -21 11.45345
	N 52° 10.369' W 021° 01.542'
  • Degrés, minutes, secondes (DD°MM'SS.s")
    • 	52 12 18.74 -21 11 27.21
	N52° 12' 18.74", W21° 11' 27.21"
	52° 12' 18.74", -21° 11' 27.21"

Numéro de référence et code de suivi
  • Numéro de référence par exemple : GK003F. C'est le numéro publique qui est utilisé pour faire référence à un GeoKret. Il est constitué du préfixe 'GK' et de 4 caractères hexadécimaux. Vous pouvez l'utiliser pour rechercher un GeoKret ou créer un lien vers ce GeoKret. Il est visible par tout le monde.
  • Tracking Code par exemple : XF3ACS. C'est un code d'identification secret indispensable pour enregistrer la "prise", le "dépôt", ou autres actions sur un GeoKret. Seul le propriétaire et les géocacheurs qui l'ont trouvé doivent le connaître. Il n'est pas recommandé de le divulguer (y compris sur les photos du GeoKret). Bien sûr, il doit être reporté sur l'étiquette du GeoKret afin de permettre son suivi.
Altitude et code des pays

SRTM et GTOPO30 fournissent des données d'élévation et permettent la conversion de coordonnées en code pays (geonames). Ainsi à chaque étape du voyage d'un GeoKret sont assignés une altitude et un code pays (son drapeau). Grâce à ces informations, sur la page de chaque GeoKret, un profil d'altitude est dessiné pour l'ensemble de son voyage. Ces informations sont calculées toutes les 10 à 15 minutes.

QR Code

Afin de faciliter les différentes actions sur le GeoKrety lors de son voyage, j'ai ajouté la possibilité d'ajouter un QR Code sur son étiquette. Le QR Code est une sorte de code-barre qui peut être "scanné" par un smartphone compatible. Ce "scan" redirigera directement sur la page du GeoKret sur geokrety.org. Vous n'aurez pas à entrer le Tracking Code, les différents codes sont encryptés dans le QR Code. En savoir plus sur les QR Code.

Un QR code encrypté avec l'adresse de la page GeoKrety.org.

Quand vous trouvez un GeoKret avec une étiquette contenant un QR Code vous pouvez :

  1. Scanner le QR Code
  2. Décoder et cliquer sur le lien qui s'affiche
  3. Enregistrer le signalement du GeoKret. Le Tracking Code sera indiqué par l'URL.

Protection des données personnelles

Votre adresse e-mail ne sera pas divulguée à des tiers. Votre mot de passe est chiffré à sens unique, il ne peut donc être révélé.

Politique de gestion des cookies

Ce site utilise des cookies pour stocker des informations nécessaires afin utiliser le site de façon convenable. Nous utilisons des cookies pour : 1) stocker les informations de connexion (vérifier l'utilisateur connecté) et 2) stocker les préférences de langue. Les deux sont absolument inoffensifs. C'est tout ! :)

Toutefois, si vous le souhaitez, vous pouvez modifier vos paramètres de cookie à tout moment dans votre navigateur.
