GeoKrety (singular: GeoKret) (From Greek geo = 'earth', and Polish krety = 'moles') is a service similar to TravelBug(TM) or GeoLutins, aiming at tracking objects you leave in geocache containers.

People move these registered objects (called GeoKrety) from cache to cache and register each move with this service. The trip of each GeoKret is then presented on a map.

GeoKrety are not a trade item, you do not have to put something in the cache to take a GeoKret, and you do not need to take something from a cache in trade for a GeoKret. If you want to take something from a cache, you should make a trade, you can place a GeoKret in addition.

Free Service: The service is absolutely free (contrary to TravelBug) and easy to use. A GeoKret can be anything, but in most cases it is a small puppet.

Please see (and edit, if necessary) the article about GeoKrety at Wikipedia .

  1. Take a card (which can be a postcard, but also a home-made card) and write a few lines for the receiver on it.
  2. As the recipient, address the Username of the recipient, their region (country, district, city) and give a target cache (best with all available waypoints - GC-code, OC-code, etc.).
  3. Create a GeoKret with the type "KretyPost", print a stamp with the Tracking Code on it and put this stamp on the postcard. (you can generate a simple stamp in the label generator on the GeoKret page).
  4. (optional:) Laminate the finished postcard to protect it during the journey.
  5. Put the card into a cache near your home-location, log the GeoKret as dropped and wait... :-)
  6. (optional:) Inform the recipient, they would probably like to observe their GeoKret.
Make your own GeoKrety
  1. Have an account at GeoKrety.org.
  2. Register the GeoKret and note down the GeoKret Tracking Code generated by the system (e.g. {$gk_example_1}). This code is secret and only you and people who find this GeoKret should know it.
  3. Print a label with the GeoKret's name, Tracking Code, Reference Number (e.g. GK003F) and owner on it. Attach this label to GeoKret (e.g. a puppet). You can use our label creator for this.

How to log a GeoKret drop in a cache from geocaching.com?

You can easily use GeoKrety with caches from geocaching.com. The separate waypoints from multicaches are not fully supported: (Licensee shall not remarket, resell, and/or redistribute the Data or any derived portion(s) of the Data in its digital form to any third party whatsoever‚Ä¶ from their waypoint license agreement ). But this is no real obstacle.

There are three options:

  • 1. Using c:geo

    c:geo is an excellent open-source Android app with support for all known geocaching websites, as well as GeoKrety!

    Rather than repeating information, take a look at their excellent user guide on this topic here. Using c:geo, you can automatically Dip the GeoKrety you have with you in all Geocaches you log!

  • 2. Manually, using copy and paste

    Go to the Log a GeoKret page. Fill in the Tracking Code of the GeoKret you wish to log, and under the section New location enter the following:

    1. In the waypoint field, enter the GC Code of the cache (eg. GC2586K).
    2. Optionally, change the Coordinates if they are different from the cache listing. See here for acceptable formats.

Reference Number and Tracking Code

Reference Number

e.g. GK003F

This is a public number that may be used to refer a GeoKret. It consists of the 'GK' prefix and 4 to 5 digits in the hexadecimal system. You can use it to search for a specific GeoKret or to link to a GeoKret. It is visible to everyone.

Tracking Code

e.g. {$gk_example_1}

This is a secret identification code which is required to post logs such as "Grab", "Drop" and others for a GeoKret. Only the owner and geocachers who found the GeoKret should know this code. This way, you need to have physically found the GeoKret in order to log it.

It is not recommended to post this code anywhere. Of course, this code must be written on a GeoKret's tag to allow finders to log it.

Altitude and country code

Thanks to SRTM and GTOPO30 elevation data and lat/lon to country code conversion (geonames), each trip point has assigned an altitude and country code (a flag). According to that data, the altitude profile is drawn. Above data and plots are updated every 10-15 minutes.

QR Code

For easy logging dropping or grabbing during your trip, I've added the possibility of attaching QR code on the GK label. Read more about QR Code. QR Code is a kind of bar code, which can be "scanned" by your mobile phone, which will redirect you to the proper page at geokrety.org ; you will not have to enter the Tracking Code, everything will be encrypted in the QR.

The QR code with encrypted address of geokrety.org page.

When you find a GeoKret with QR code on the label you can:

  1. Take a photo of the code with your mobile phone
  2. Decode it and click on the url
  3. Log the grabbing or hiding the GeoKret. The Tracking Code will be indicated by the url.
