Over GeoKrety

GeoKrety (uit het Grieks geo = â€˜aardeâ€™, en Pools krety = â€˜mollenâ€˜) is een service gelijk aan de Travelbug(TM) of GeoLutins, met de bedoeling objecten te volgen die je achterlaat in caches. Mensen verplaatsen deze geregistreerde objecten (genoemd GeoKrety) van cache naar cache en registreren elk verplaatsing met deze service. De reis van elke GeoKret wordt dan gepresenteerd op een kaart.

Geokrety zijn geen ruilobjecten, je hoeft niets in de cache te plaatsen als je een Gk meeneemt en je hoeft niets te ruilen als je een Gk achterlaat in een cache. Als je iets meeneemt uit een cache, moet je iets in ruil achterlaten. Je kan daarna ook een GeoKret plaatsen.

De service is absoluut gratis (in tegenstelling tot de TravelBug) en gemakkelijk te gebruiken. Een GeoKret kan van alles zijn, maar in de meeste gevallen is het een kleine poppetje.

Lees ook (en bewerk indien nodig) het artikel over geokrety op Wikipedia.

Hoe spreek je geokrety uit? ;)

Maak je eigen GeoKrety
  1. Maak een account aan op GeoKrety.org service
  2. Registreer een GeoKret en noteer de GeoKret Tracking Code (bv XF3ACS) welke gegenereerd is door het systeem. Deze code is geheim en alleen u en mensen die dit vinden, kennen deze.
  3. Create a GeoKret with type "KretyPost", print a stamp with the Tracking Code and put this stamp on the postcard. (u kunt een eenvoudige stempel genereren in label generator op de GeoKret pagina)

Dan moet je:

  1. Laat je GeoKrety achter in een cache en log de locatie op deze service
  2. Veel plezier met volgen van de reis die deze GeoKret maakt!

KretyPost regels
Korte beschrijving:
  1. Neem een kaart (bijvoorbeeld een briefkaart, maar kan ook een zelfgemaakte kaart zijn) en schrijf een paar regels voor de ontvanger.
  2. Als ontvangst adres de bijnaam van de ontvanger, de regio (land, district, stad) en geef een doelcache (het beste met alle beschikbare waypoints - GC-code, OC-code, etc.).
  3. Maak een GeoKret met type "KretyPost", print een stempel (zie voorbeelden) met de Tracking Code en zet deze postzegel op de ansichtkaart. (u kunt een eenvoudige stempel genereren in label generator op de GeoKret pagina)
  4. (optioneel:) Lamineer de afgewerkte briefkaart om het te beschermen tijdens de reis.
  5. Plaats de kaart in een cache dichtbij uw thuislocatie, log de GeoKret als dropped en wacht ... :-)
  6. (optioneel:) Informeer de ontvanger, hij zal waarschijnlijk deze GeoKret willen volgen

Referentienummer en de Tracking Code

Referentienummer

GK003F

Het is een publiek nummer dat kan worden gebruikt om naar een GeoKret te verwijzen. Het bestaat uit 'GK' voorvoegsel en 4 cijfers in hexadecimaal systeem. Je kunt het gebruiken om specifieke GeoKret te zoeken of om te linken naar deze GeoKret. Het is zichtbaar voor iedereen.

Tracking Code

XF3ACS

Het is een geheime identificatiecode die nodig is om logs te plaatsen zoals "Grab", "Drop" en anderen voor een GeoKret. Alleen de eigenaar en geocachers die de GeoKret hebben gevonden moeten deze code kennen. Het is niet aangeraden om het ergens te plaatsen. Natuurlijk moet deze code worden geschreven op de tag van GeoKret, om het mogelijk te maken dat vinders loggen.

Hoogte en land code

Dankzij SRTM en GTOPO30 hoogtegegevens en lat/lon naar landcode conversie (geonames), elk reispunt heeft een hoogte- en landcode (een vlag) toegewezen. Volgens die gegevens wordt het hoogte profiel getrokken. Boven gegevens en plots worden elke 10-15 minuten bijgewerkt.

QR Code

Ik heb de mogelijkheid toegevoegd om QR-code toe te voegen aan het GK-label, voor het makkelijk ophangen of meenemen van je reis. Lees meer over QR Code. Lees meer over QR Code.

De QR-code met gecodeerde adres van geokrety.org pagina.

Wanneer je een GeoKret met QR-code op het label vindt kunt je het volgende doen:

  1. Neem een foto van de code met de mobiele telefoon
  2. Decoderen en klik op de url
  3. Log het meenemen of achterlaten van de GeoKret. De Tracking Code wordt aangegeven door de url.
