Over GeoKrety

GeoKrety (uit het Grieks geo = ‘aarde’, en Pools krety = ‘mollen‘) is een service gelijk aan de Travelbug(TM) of GeoLutins, met de bedoeling objecten te volgen die je achterlaat in caches. Mensen verplaatsen deze geregistreerde objecten (genoemd GeoKrety) van cache naar cache en registreren elk verplaatsing met deze service. De reis van elke GeoKret wordt dan gepresenteerd op een kaart.

Geokrety zijn geen ruilobjecten, je hoeft niets in de cache te plaatsen als je een Gk meeneemt en je hoeft niets te ruilen als je een Gk achterlaat in een cache. Als je iets meeneemt uit een cache, moet je iets in ruil achterlaten. Je kan daarna ook een GeoKret plaatsen.

De service is absoluut gratis (in tegenstelling tot de TravelBug) en gemakkelijk te gebruiken. Een GeoKret kan van alles zijn, maar in de meeste gevallen is het een kleine poppetje.

Lees ook (en bewerk indien nodig) het artikel over geokrety op Wikipedia.

Hoe spreek je geokrety uit? ;)

Maak je eigen GeoKrety
  1. Maak een account aan op GeoKrety.org service
  2. Registreer een GeoKret en noteer de GeoKret Tracking Code (bv XF3ACS) welke gegenereerd is door het systeem. Deze code is geheim en alleen u en mensen die dit vinden, kennen deze.
  3. Print een label met de GeoKret's naam, Tracking Code, Referentie Nummer (bv GK003F) en de naam van de eigenaar. Voeg dit label toe aan je GeoKret (bijv. een poppetje)

Dan moet je:

  1. Laat je GeoKrety achter in een cache en log de locatie op deze service
  2. Veel plezier met volgen van de reis die deze GeoKret maakt!

KretyPost regels
Korte beschrijving:
  1. Neem een kaart (bijvoorbeeld een briefkaart, maar kan ook een zelfgemaakte kaart zijn) en schrijf een paar regels voor de ontvanger.
  2. Als ontvanger adres de bijnaam van de ontvanger, de regio (land, district, stad) en geef een doelcache (het beste met alle beschikbare waypoints - GC-code, OC-code, etc.).
  3. Maak een GeoKret met type "KretyPost", print een postzegel af (zie voorbeelden) met de tracking code en plak de postzegel op de ansichtkaart. (u kunt een eenvoudige stempel genereren in label generator op de GeoKret pagina)
  4. (optioneel:) Lamineer de afgewerkte briefkaart om het te beschermen tijdens de reis.
  5. Plaats de kaart in een cache dichtbij uw thuislocatie, log de GeoKret als dropped en wacht ... :-)
  6. (optioneel:) Informeer de ontvanger, hij zal waarschijnlijk deze GeoKret willen volgen

Hoe het achterlaten loggen in een geocaching.com cache?

GC waypoints worden niet volledig ondersteund door GeoKrety Licentiehouder zal de data niet commercialiseren, verkopen, en/of herverdelen noch afgeleide gegevens in zijn digitale vorm aan eender welke derde partij… van waypoint licentieovereenkomst. U kunt echter gemakkelijk GeoKrety gebruiken met GC caches.

Er zijn twee opties:

  • De eenvoudigste en aanbevolen manier is om onze GeoKrety Toolbox (GKT) te gebruiken

    Dit is een greasemonkey script (geschreven en onderhouden door simor).

Referentienummer en Tracking Code

Referentienummer

bijv. GK003F

Het is een publiek nummer dat kan worden gebruikt om naar een GeoKret te verwijzen. Het bestaat uit 'GK' voorvoegsel en 4 cijfers in hexadecimaal systeem. You can use it to search a specific GeoKret or to link to this GeoKret. Het is zichtbaar voor iedereen.

Tracking Code

e.g. XF3ACS

It is a secret identification code which is required to post logs such as "Grab", "Drop" and others for a GeoKret. Only the owner and geocachers who found the GeoKret should know this code. It is not recommended to post it anywhere. Of course this code must be written on GeoKret's tag to allow finders to log it.

Hoogte en land code

Thanks to SRTM and GTOPO30 elevation data and lat/lon to country code conversion (geonames), each trip point has assigned an altitude and country code (a flag). Volgens die gegevens wordt het hoogte profiel getrokken. Above data and plots are updated every 10-15 minutes.

QR Code

For easy logging dropping or grabbing during your trip, I've added the possibility of attaching QR code on the GK label. QR Code is a kind of bar code, which can be "scanned" by your mobile phone, which will redirect you to the proper page at geokrety.org ; you will not have to enter the tracking code, everything will be encrypted in the QR. Read more about QR Code.

De QR-code met gecodeerde adres van geokrety.org pagina.

Wanneer je een GeoKret met QR-code op het label vindt kunt je het volgende doen:

  1. Neem een foto van de code met de mobiele telefoon
  2. Decoderen en klik op de url
  3. Log het meenemen of achterlaten van de GeoKret. De trackingcode wordt aangegeven door de url.

How to decode?

You can use the free software for your mobile phone, eg: Barcode Scanner

HTTPS en SSL

The connection to our server is secured using TLS encryption. U zou een groen slot moeten zien in uw navigatieadresbalk. Het geeft aan dat de communicatie tussen u en de server niet kan worden gelezen door anderen en dit is prima voor uw inloggegevens!
