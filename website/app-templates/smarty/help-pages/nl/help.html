Over GeoKrety

GeoKrety (singular: GeoKret) (From Greek geo = 'earth', and Polish krety = 'moles') is a service similar to TravelBug(TM) or GeoLutins, aiming at tracking objects you leave in geocache containers.

People move these registered objects (called GeoKrety) from cache to cache and register each move with this service. De reis van elke GeoKret wordt dan gepresenteerd op een kaart.

GeoKrety are not a trade item, you do not have to put something in the cache to take a GeoKret, and you do not need to take something from a cache in trade for a GeoKret. If you want to take something from a cache, you should make a trade, you can place a GeoKret in addition.

Free Service: The service is absolutely free (contrary to TravelBug) and easy to use. Een GeoKret kan van alles zijn, maar in de meeste gevallen is het een kleine poppetje.

Please see (and edit, if necessary) the article about GeoKrety at Wikipedia .

KretyPost regels
Korte beschrijving:
  1. Neem een kaart (bijvoorbeeld een briefkaart, maar kan ook een zelfgemaakte kaart zijn) en schrijf een paar regels voor de ontvanger.
  2. As the recipient, address the Username of the recipient, their region (country, district, city) and give a target cache (best with all available waypoints - GC-code, OC-code, etc.).
  3. Create a GeoKret with the type "KretyPost", print a stamp with the Tracking Code on it and put this stamp on the postcard. (you can generate a simple stamp in the label generator on the GeoKret page).
  4. (optioneel:) Lamineer de afgewerkte briefkaart om het te beschermen tijdens de reis.
  5. Put the card into a cache near your home-location, log the GeoKret as dropped and wait... :-)
  6. (optional:) Inform the recipient, they would probably like to observe their GeoKret.
Maak je eigen GeoKrety
  1. Have an account at GeoKrety.org.
  2. Register the GeoKret and note down the GeoKret Tracking Code generated by the system (e.g. {$gk_example_1}). Deze code is geheim en alleen u en mensen die dit vinden, kennen deze.
  3. Print a label with the GeoKret's name, Tracking Code, Reference Number (e.g. GK003F) and owner on it. Attach this label to GeoKret (e.g. a puppet). You can use our label creator for this.

Dan moet je:

  1. Put your GeoKret in a cache and register its location in the service.
  2. Veel plezier met volgen van de reis die deze GeoKret maakt!

How to log a GeoKret drop in a cache from geocaching.com?

You can easily use GeoKrety with caches from geocaching.com. The separate waypoints from multicaches are not fully supported: (Licensee shall not remarket, resell, and/or redistribute the Data or any derived portion(s) of the Data in its digital form to any third party whatsoever… from their waypoint license agreement ). But this is no real obstacle.

There are three options:

  • 1. Using c:geo

    c:geo is an excellent open-source Android app with support for all known geocaching websites, as well as GeoKrety!

    Rather than repeating information, take a look at their excellent user guide on this topic here. Using c:geo, you can automatically Dip the GeoKrety you have with you in all Geocaches you log!

  • 2. Handmatig, met behulp van kopie en plak

    Go to the Log a GeoKret page. Fill in the Tracking Code of the GeoKret you wish to log, and under the section New location enter the following:

    1. In the waypoint field, enter the GC Code of the cache (eg. GC2586K).
    2. Optionally, change the Coordinates if they are different from the cache listing. See here for acceptable formats.

Referentienummer en de Tracking Code

Referentienummer

GK003F

This is a public number that may be used to refer a GeoKret. It consists of the 'GK' prefix and 4 to 5 digits in the hexadecimal system. You can use it to search for a specific GeoKret or to link to a GeoKret. Het is zichtbaar voor iedereen.

Tracking Code

e.g. {$gk_example_1}

This is a secret identification code which is required to post logs such as "Grab", "Drop" and others for a GeoKret. Alleen de eigenaar en geocachers die de GeoKret hebben gevonden moeten deze code kennen. This way, you need to have physically found the GeoKret in order to log it.

It is not recommended to post this code anywhere. Of course, this code must be written on a GeoKret's tag to allow finders to log it.

Hoogte en land code

Dankzij SRTM en GTOPO30 hoogtegegevens en lat/lon naar landcode conversie (geonames), elk reispunt heeft een hoogte- en landcode (een vlag) toegewezen. Volgens die gegevens wordt het hoogte profiel getrokken. Boven gegevens en plots worden elke 10-15 minuten bijgewerkt.

QR Code

Ik heb de mogelijkheid toegevoegd om QR-code toe te voegen aan het GK-label, voor het makkelijk ophangen of meenemen van je reis. Lees meer over QR Code. Lees meer over QR Code.

De QR-code met gecodeerde adres van geokrety.org pagina.

Wanneer je een GeoKret met QR-code op het label vindt kunt je het volgende doen:

  1. Neem een foto van de code met de mobiele telefoon
  2. Decoderen en klik op de url
  3. Log het meenemen of achterlaten van de GeoKret. De Tracking Code wordt aangegeven door de url.

Persoonlijke gegevens bescherming

Uw e-mailadres wordt niet overgedragen aan derden. Uw wachtwoord is eenrichtings versleuteld, zodat het niet kan worden onthuld.

Cookies beleid

We gebruiken cookies enkel om er zeker van te zijn dat onze website goed voor u werkt. Deze Cookie blijft op uw computer voor:

  • standaard, totdat u uw browser afsluit.
  • of {GK_SITE_SESSION_LIFETIME_REMEMBER/86400} dagen als je de optie onthoudt mij wanneer je inlogt.

Het is absoluut ongevaarlijk. Dat is het! 😀

Cookies worden automatisch vernietigd wanneer je expliciet uitlogt. Als je echter wilt, kun je op elk gewenst moment je cookie instellingen in je browser wijzigen.

HTTPS en SSL

De verbinding met onze server is beveiligd met TLS-versleuteling. U zou een groen slot moeten zien in uw navigatieadresbalk. Het geeft aan dat de communicatie tussen u en de server niet kan worden gelezen door anderen en dit is prima voor uw inloggegevens!

Onze server gebruikt IPv6, wat betekent dat we klaar zijn voor de toekomst! IPv6: De toekomst is nu meer dan ooit
